### ET外发平台功能

> 文件传输流程
>
> 1. 登录 文件交换系统 账号
> 2. 添加 “合作伙伴”
> 3. 新增"项目管理"
> 4. 通过 “文件外发”、“版型销售” 把文件外发到合作方
>    - 文件外发时接收方仅可下载、查看、编辑，不可外发，文件所有权在发送方
>    - 版型销售时接收方可下载、查看、编辑、外发，文件所有权在接收方。
> 5. 合作方通过“需要下载”列表中查看发送方传来的文件&CAD工具中拉取文件
> 6. 接收方编辑修改文件后可通过CAD工具中“回传”功能回传文件。

### 文件外发流程

#### 文件外发功能规则：

1. **通过判断乙方在我们这边注册的CAD版本来确定甲方能发送什么文件。**
2. **不会出现一个公司有多个版本的情况**
   1. O开头的是ETS线上用户
   2. L开头的是ETS线下用户
   3. P开头的是RPJ用户
   4. D开头的是意向客户(盗版)
3. **通过文件类型划分：**

   - PRJ
   - 线上ETS
     - 线上ETS发给线下ETS不支持丙方
   - 线下ETS
   - 盗版文件
     - 盗版文件外发不需要进行转换；只能发给线下用户(RPJ、ETS)，不可发给线上用户。
     - 盗版文件转正版文件不需要收费
     - 正版文件转盗版文件需要收费
   - 其他文件
     - TXT
     - PDF
     - 仅双方为ETS线下版本支持
4. **通过发送身份划分：**

   - 甲方
     - 发送给乙方
     - 接收乙方
   - 乙方
     - 发送给甲方/丙方
     - 接收甲方/丙方
   - 丙方
     - 仅可发送给乙方
     - 仅可接收乙方
5. **通过文件权限划分：**

   - 不可下载、查看---取消合作伙伴关系

   - 只可下载，查看、修改---建立合作伙伴关系
   - 可查看可修改可转发(转发为丙方)----甲方开启丙方权限
   - 授权该供应商打开本公司文件开关---可通过其他方式发送文件，不需通过外发文件功能。
   - 拥有文件所有权---版型销售
6. **4个维度**

   1. 谁发给谁？
      1. ETS线上用户
         1. 发给线上用户---只可发送ETS
         2. 发给线下用户---只可发送ETS
         3. 发给PRJ用户---不可外发，只可只可开启授权后通过其他渠道进行外发。
         4. 发给盗版用户---不可外发，只可开启授权后通过其他渠道进行外发。
      2. ETS线下用户
         1. 发给线下用户---只可发送ETS
         2. 发给线上用户---只可发送ETS
         3. 发给PRJ用户---发给PRJ用户时会转换为ODP格式，回传时依旧是ODP格式
         4. 发给盗版用户---只可发送ETS，会转换为PRJ
      3. PRJ用户
         1. 发给PRJ用户---只可发送PRJ
         2. 发给线上用户---只可发送PRJ，会转换为ETS
         3. 发给线下用户---只可发送PRJ，会转换为ETS
         4. 发给盗版用户---只可发送PRJ
      4. 意向用户(盗版)
         1. 发给盗版用户---无需通过外发平台。
         2. 发给线上用户---不可外发，不可授权。
         3. 发给线下用户---只可发送PRJ，会转换为ETS
         4. 发给PRJ用户---只可发送PRJ
      5. 其他平台用户
         1. 不通过ET外发平台
   2. 我发的什么文件？
      1. ETS
         1. ETS版本只能发ETS文件
      2. PRJ
         1. PRJ版本只能发PRJ文件
      3. 盗版文件
         1. 正版用户发送盗版文件，意向用户下载时扣取费用
         2. 意向用户发送盗版文件，正版用户下载时扣取意向用户费用
      4. DXF文件---通过其他渠道外发即可
   3. 通过什么方式发？
      1. 线上用户仅可通过CAD外发文件
      2. 线下用户仅可通过平台外发文件
      3. 可开启授权该供应商打开本公司文件开关---可通过其他方式发送文件，不需通过外发文件功能。
   4. 文件的操作权限设置？
      1. 不可下载、查看---取消合作伙伴关系
      2. 只可下载，查看、修改---建立合作伙伴关系
      3. 可查看可修改可转发(转发为丙方)----甲方开启丙方权限，丙方须为ETS线上用户。
      4. 拥有文件所有权---版型销售
   5. 回传
      1. 哪里来回哪里去
7. **ETS线上版本与ETS线下版本的区别**

   1. 鉴权
      1. 线上通过账号登录
      2. 线下通过锁进行登录
   2. 外发模式不同
      1. 线上通过CAD
      2. 线下通过外发平台
   3. 管理不同
      1. 线上需先添加合作伙伴并在项目管理中进行添加方可在CAD进行文件外发	

         1. 添加合作伙伴

         2. 新增项目：选择合作伙伴为接收方

         3. 新增丙方，通过项目管理中指定丙方进行新增

            - 丙方也必须为ETS线上用户

            - 乙方发送丙方文件需通过第三方

      2. 线下添加合作伙伴后在平台上直接外发文件即可

         1. 外发文件成功后会自动创建一个项目
         2. 甲方为发送方，乙方为接收方
8. **授权该供应商打开本公司的软件**

   1. 该功能使乙方通过所有渠道获得的授权甲方文件都能打开。
9. **丙方功能**

   1. 实际上是为乙方增加了一个转发的功能
      1. 转发后，丙方能否打开该文件呢？是否需要成为甲方的合作伙伴？
10. **具体流程：**
    1. 线上ETS：
       1. 文件导入：参考上面的“文件导入模块”
       2. 文件导出：参考上面的“文件导出模块”
       3. 版权销售：文件导出：参考上面的“版权销售模块”
       4. 文件外发：接收方 线上/线下 流程均一致
          1. CodeMeter上输入IP地址(线上ETS输入服务器IP)
          2. 打开文件夹内的“SetUrl”，选择环境(测试需要这一步骤，用户不用)
          3. 打开CAD进行账户登录校验
          4. 新建/导入/修改文件后保存为项目文件
             - 外发平台操作：登录账户→新建项目→选择合作伙伴→保存项目即可
          5. CAD中点击"移交"选择“合作方”即可
          6. 验证：
             1. 线上ETS：登录CAD，选择“下载文件”即可
             2. 线下ETS/PRJ：登录外发平台，“需要下载”列表选中文件下载，在CAD打开即可
    2. 线下ETS：
       1. 文件导入：参考上面的“文件导入模块”
       2. 文件导出：参考上面的“文件导出模块”
       3. 版权销售：文件导出：参考上面的“版权销售模块”
       4. 文件外发：接收方 线上/线下 流程均一致
          1. CodeMeter上输入IP地址(线下ETS输入本地IP)
          2. 插上对应账户的锁
          3. 打开CAD时会提示到期时间等信息，该信息则身份验证成功。
          4. 新建/导入/修改文件后保存为项目文件
          5. 登录外发平台：登录账号→外发文件→选择合作伙伴→选择文件
          6. 验证：
             1. 线上ETS：登录CAD，选择“下载文件”即可
             2. 线下ETS/PRJ：登录外发平台，“需要下载”列表选中文件下载，在CAD打开即可
    3. 线下PRJ：
       1. 文件导入：参考上面的“文件导入模块”
       2. 文件导出：参考上面的“文件导出模块”
       3. 版权销售：文件导出：参考上面的“版权销售模块”
       4. 文件外发：接收方 线上/线下 流程均一致
          1. CodeMeter上输入IP地址(线下ETS输入本地IP)
          2. 插上对应账户的锁
          3. 打开CAD时会提示到期时间等信息，该信息则身份验证成功。
          4. 新建/导入/修改文件后保存为项目文件
          5. 登录外发平台：登录账号→外发文件→选择合作伙伴→选择文件
          6. 验证：
             1. 线上ETS：登录CAD，选择“下载文件”即可
             2. 线下ETS/PRJ：登录外发平台，“需要下载”列表选中文件下载，在CAD打开即可





规则点：

1. 线下ETS：
   - 文件外发后自动创建项目
   - 项目名称=文件保存时的样板号名称
   - 项目内可以有多个乙方，但项目指定乙方仅为第一个乙方。其他乙方看不到该项目。
2. 线上ETS：
   - 创建项目后，即使与乙方/丙方脱离“合作伙伴”关系，乙方&丙方依旧可以打开项目内的文件并修改。
     - 文件能否打开，仅限于项目状态是否为完结。
3. 线下PRJ：
   - PA与PB互为合作伙伴，PB版型销售中找不到PA
   - 同一个文件 先高后低 依旧提示没有权限
4. 意向用户：
   1. 接收来自线下ETS外发文件，下载时不扣费。只限ETS用户，PRJ用户依旧扣费。
   2. 外发文件给线下ETS用户，转换给LB扣除1元，转换给LA扣除10元
   3. 意向用户外发文件给接收方时，接收方如有折扣，则接收方在下载文件时意向用户会享受该折扣
